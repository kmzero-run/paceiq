Cómo configurar en 10 minutos
1
Sube este archivo a GitHub Pages
Crea un repositorio en github.com → sube este archivo como index.html → activa GitHub Pages en Settings → Pages. Tu URL será https://tuusuario.github.io/paceiq
2
Registra tu app en Strava
Ve a strava.com/settings/api → crea una nueva aplicación → en "Authorization Callback Domain" pon tu dominio de GitHub Pages (sin https://). Copia el Client ID y Client Secret.
3
Introduce tus credenciales arriba
Pega el Client ID y Client Secret en los campos de arriba. Se guardan únicamente en tu navegador (localStorage).
4
Obtén tu API Key de Anthropic
Regístrate en console.anthropic.com → API Keys → Create Key. Pégala en el campo de arriba.
5
Pulsa "Conectar con Strava"
Se abre el login de Strava → autorizas → vuelves a PaceIQ con tus actividades cargadas automáticamente.
